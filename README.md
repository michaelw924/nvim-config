# Michael's Neovim Config

Portable Neovim configuration based on kickstart.nvim.

## Install (Linux)

Run:

```bash
curl -fsSL https://raw.githubusercontent.com/michaelw924/nvim-config/main/install.sh | bash
```

This installs neovim, dependencies, and your config.

## Setup (already have neovim installed)

If you already have neovim installed and just want to set up your config:

```bash
curl -fsSL https://raw.githubusercontent.com/michaelw924/nvim-config/main/setup.sh | bash
